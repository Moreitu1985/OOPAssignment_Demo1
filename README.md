# C# OOP Assignment

This project is a C# console application demonstrating the four core principles of Object-Oriented Programming (OOP):

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

The assignment was implemented using clean class design, constructor validation, interfaces, inheritance hierarchies, abstract classes, and polymorphism.
